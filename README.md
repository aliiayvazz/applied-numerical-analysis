# Applied Numerical Analysis & Mathematical Modeling

A collection of Python-based numerical computing projects focused on solving complex engineering problems through mathematical modeling, algorithm design, and data visualization. These projects bridge the gap between theoretical mathematics and applied electronics engineering.

## Core Implementations & Algorithms

* **Iterative Linear Solvers (Gauss-Seidel Method):** Implemented numerical algorithms to solve large systems of linear equations. Evaluated convergence criteria and algorithm stability, which are fundamental in circuit simulations and computational electromagnetics.
* **Coordinate Transformations & Vector Math:** Programmed dynamic conversions between Polar and Cartesian coordinate systems. This mathematical foundation is directly applicable to **Radar System Tracking**, antenna radiation patterns, and autonomous navigation.
* **Scientific Data Visualization:** Utilized advanced plotting libraries to visualize mathematical functions, algorithmic outputs, and simulation data, translating raw numerical results into actionable engineering insights.

## Technology & Tools Stack
* **Language & Environment:** Python, Jupyter Notebook
* **Scientific Libraries:** NumPy, Matplotlib
* **Documentation:** PDF technical reports accompanying each computational model.

## Engineering Value for Defense & Aerospace
In aerospace and defense technologies, hardware is only as good as the mathematical models driving it. This repository demonstrates:
* The ability to translate complex mathematical theories (Numerical Analysis, Linear Algebra) into functional Python code.
* Algorithmic problem-solving skills necessary for signal processing, trajectory modeling, and sensor data analysis.
* Strong technical documentation and reporting habits.
